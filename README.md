# GeoWatch — Geopolitical World Clock

Real-time world clock focused on USA–Iran geopolitical tensions.

## Deploy to Vercel

### Alternativ 1 — Vercel CLI (snabbast)
```bash
npm i -g vercel
vercel
```

### Alternativ 2 — GitHub + Vercel Dashboard
1. Push projektet till ett GitHub-repo
2. Gå till vercel.com → "Add New Project"
3. Importera repot — Vercel känner igen Vite automatiskt
4. Klicka "Deploy" — klart!

## Lokal utveckling
```bash
npm install
npm run dev
```
